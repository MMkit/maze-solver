Simulation Interface

The side panel provide the user interface for the mouse animation.

The features are:

  • Available Mazes
    • Here the map is selected.
  • Available Algorithms
    • Here the algorithm is selected.
  • Simulation Graphics Options
    • Fog - shows a fog on the path the mouse does not see.
    • Path - shows a green path for current, blue for first path, and red for best path
    • Info - shows information on how the algorithm works
    • Classic - shows a classic map style.
  • Simulation Speed
    • use to control how face the mouse travels.
  • Simulation controls
    • Stop - halts the simulation and set mouse back to start.
    • Pause - pauses the simulation.
    • Start - starts the simulation where it left off.
