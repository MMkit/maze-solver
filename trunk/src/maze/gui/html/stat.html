Statistics Display

In the Statistics Display, you are able to pick the maze and the algorithm to see the relevant statistics. On the left is the map of the path taken.

The relevant statistics are:

  • Number of Unique Squares Traversed
  • Number of Cells Traversed to Reach the Center Once
  • Number of Turns Taken
  • Number of Cells Traversed on Best Run
  • Number of Turns Taken
  • Total Number of Cells Traversed to Complete Best Run
  • Number of Turns Taken
